# 新增工具：write_file、grep、list_files、web_fetch

**日期**: 2026-06-15
**状态**: 已实现
**前置 spec**: [Mini Agent Loop 设计](./2026-06-14-mini-agent-loop-design.md)

## 概述

参考 Claude Code 的工具集，新增 4 个工具，补齐文件操作闭环和网络能力。

## 新增工具

| 工具 | 功能 | 安全约束 |
|------|------|----------|
| `write_file` | 写入/创建文件 | 限制项目目录内，拒绝覆盖 .env/.git 等敏感路径 |
| `grep` | 正则搜索文件内容 | 限制项目目录内，超时 10s |
| `list_files` | 列出目录结构 | 限制项目目录内，最大深度 3 层 |
| `web_fetch` | 抓取网页内容并提取文本 | 超时 15s，内容截断 50KB |

## 实现方式

### write_file
- 参数：`path`（文件路径）、`content`（写入内容）
- 路径遍历防护（同 read_file）
- 拒绝写入：`.env`、`.git/`、`venv/` 等敏感路径
- 自动创建父目录
- 返回成功或错误信息

### grep
- 参数：`pattern`（正则或关键字）、`path`（目录或文件，默认当前目录）
- 用 Python `re` 模块匹配
- 限制搜索范围在项目目录内
- 结果截断到 20 条匹配

### list_files
- 参数：`path`（目录路径，默认当前目录）、`depth`（最大深度，默认 2）
- 用 `os.walk` 遍历
- 忽略 `.git`、`venv`、`__pycache__`、`.egg-info` 等
- 返回树状结构文本

### web_fetch
- 参数：`url`（网页地址）
- 用 `requests` 抓取
- 用 `html.parser` 或正则提取纯文本
- 内容截断到 50KB

## 影响范围

- `tools.py`：新增 4 个工具函数 + 4 个 schema + 注册
- 其他文件无需修改（工具注册表自动生效）
